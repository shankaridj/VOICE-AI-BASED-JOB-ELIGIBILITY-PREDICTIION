# VOICE-AI-BASED-JOB-ELIGIBILITY-PREDICTIION
An AI-powered recruitment assistance system that evaluates a candidate's communication skills through voice analysis and predicts job eligibility for a selected domain using Machine Learning techniques. The project integrates speech processing, feature extraction, and predictive analytics to support intelligent career assessment.

This project is an AI-powered job eligibility prediction system that analyzes a candidate's voice and communication skills to determine their suitability for a selected job domain.

The user first selects a preferred job domain and then provides a voice response. The system extracts speech and communication-related features, processes them using Machine Learning algorithms, and predicts whether the candidate is eligible for the chosen domain.

## Key Features
- Domain-based eligibility prediction
- Voice recording and analysis
- Communication skill assessment
- Speech feature extraction
- Machine Learning-based prediction
- Interactive web interface
